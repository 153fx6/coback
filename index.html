🐔코코포 백업용 로그서식 변환기

코코포리아에서 출력한 html로그 파일에만 대응합니다.
3번의 필수 입력 사항을 제외한 항목은 모두 선택사항이므로 비워두셔도 정상작동합니다.
자세한 사용법 안내는 하단의 배포링크를 확인해주세요.

1. 세션 설정

세션카드 너비는 px과 %단위를 지원합니다. (공란으로 둘시 100%너비로 출력됩니다.)

2. 캐릭터 설정

이미지나 색상을 추가로 지정할 경우에만 입력.

* 중복된 이름의 캐릭터는 마지막에 설정된 정보가 우선 적용됩니다.

3. 채팅로그 파일 불러오기*

다중파일 업로드 시 중복되는 내용을 이어붙여 하나의 파일로 병합합니다.
파일을 드래그하여 병합 순서를 변경할 수 있습니다.

채팅로그 파일을 이곳에 드래그하거나 클릭하세요.

    4. 디자인 설정

    로그 스타일을 선택할 수 있습니다.

    * 커스텀 파일(css) 다운로드는 하단의 배포링크 참고.

    업로드한 CSS 파일이 로그에 직접 삽입됩니다.
